== Description ==
This Plugin have a youtube like default view and a popup view to meet your requirement

== Installation ==
1. Upload the entire `nabz-image-gallery` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

3.After activation you can upload new images into your gallery from the upload new link. 
You can also edit existing images and its associated descriptions from the edit image 
link in the Plugin home page and also delete a image from there.

4.please change the file permission of '/wp-content/plugins/nabz-image-gallery/images' to 777.thats it

5.Now just add a new page or post in your blog and put "[view_gallery]" tag into its 
content area thats it.Go to front page to see the Plugin working.

        go to Plugin help page for more information
